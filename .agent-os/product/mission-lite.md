# Product Mission (Lite)

Venture Onboarding System is an agent module + skill suite built into SB-OS that automatically scaffolds any new venture with a tailored launch checklist, Google Drive folder structure, and AI-powered specialist agents.

It serves Sayed as a solo multi-venture operator who needs every venture to start with the same strong operational foundation. Unlike manual setup or generic templates, the system detects venture type, filters the universal launch checklist to only relevant tasks, creates them directly in SB-OS, and dispatches specialist skills (brand, legal, content, offer) that each own their domain end-to-end — from research to draft to Drive output to task completion.
